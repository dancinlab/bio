# hexa-bio 신약발견 리포트 (쉽게 풀어쓴 버전)

작성: 2026-05-08

## 한 줄 요약

**컴퓨터로 양자물리 계산을 써서 200가지 질병의 신약 후보 200개를 설계한 9개월 프로젝트.**

---

## 1. 무엇을 하는 프로젝트인가요?

신약을 만들려면 보통 분자 한 개가 인체 안 단백질에 얼마나 잘 "달라붙는지" 계산해야 합니다. 이걸 정확히 알려면 양자역학 계산이 필요한데, 너무 무거워서 보통은 근사치만 씁니다.

**hexa-bio**는 **양자 컴퓨터의 알고리즘(VQE)** 을 빌려와 이 계산을 더 정확하게 하려는 프로젝트입니다. 5가지 도구(축)로 신약을 설계합니다:

| 축 | 비유 | 무엇을 하는가 |
|---|---|---|
| **QUANTUM** | 망원경 | 분자가 표적에 어떻게 결합하는지 양자 계산 |
| **WEAVE** | 운반차 | 약물을 몸 안 정확한 위치로 배달 (LNP, ADC 링커 등) |
| **VIROCAPSID** | 트로이 목마 | 바이러스 껍질로 유전자 치료 운반 (AAV) |
| **RIBOZYME** | 메모 차단기 | RNA를 이용해 잘못된 단백질 생산 차단 (siRNA, ASO) |
| **NANOBOT** | 미세 로봇 | 나노 입자로 표적 세포 공략 |

---

## 2. 9개월간 무엇을 했는가?

200가지 질병에 대해 각각:
- **현재 표준 치료제 30개 정도** 정리
- **2024년 신규 FDA 승인 약** 핵심 정리
- **양자-VQE 가 어디 적용 가능한지** 식별
- **새로운 약물 후보 1개** 양자-VQE 설계 (`hxq-*-001`)

**총 200개 신약 후보가 in-silico (컴퓨터상)로 설계됨.**

## 4. 주요 신약 후보 (가장 흥미로운 것들)

### 🎯 Cancer

- **`hxq-aml-menin-001`** — 백혈병 신약. 2024 FDA 승인된 revumenib 보다 더 넓은 환자군 cover.
- **`hxq-pdac-krasg12d-001`** — 췌장암 90% 환자가 가진 KRAS G12D 변이 표적. 역사적으로 "약 못 만드는 표적"이었음.
- **`hxq-bc-her2low-001`** — 유방암 HER2-low 환자 대상 차세대 ADC.

### 🎯 Neuro

- **`hxq-pd-lrrk2-001`** — 파킨슨 병 LRRK2 G2019S (한국·아슈케나지 호발) 표적.
- **`hxq-dem-abeta-002`** — 알츠하이머 차세대 항-Aβ. ARIA 부작용 회피.
- **`hxq-stroke-tnk-002`** — 뇌졸중 치료제 tenecteplase 보다 출혈 적음.

### 🎯 Cardio + Metabolic

- **`hxq-htn-agt-001`** — 고혈압 분기 1회 주사 (siRNA).
- **`hxq-mash-thrb-002`** — 지방간염 (resmetirom 2024 FDA 후속).
- **`hxq-osa-glp1-001`** — 수면무호흡 + 비만 동시 약 (tirzepatide 2024 후속).

### 🎯 Modality-platform

- **`hxq-gtp-aav-001`** — 차세대 AAV 캡시드. 면역 반응 회피 + CNS+근육 동시 표적.
- **`hxq-tpd-arv-001`** — PROTAC 차세대. 유방암+전립선암 동시.
- **`hxq-aidd-protein-001`** — AlphaFold + 양자-VQE 통합 단백질 설계.

### 🎯 한국 K-Pharma

- **`hxq-kpharma-platform-001`** — 한국 식약처 + 글로벌 동시 승인 path.
- **`hxq-tkm-ginseng-001`** — 인삼 사포닌 (ginsenoside Rg3) 면역조절제.
- **`hxq-gerd-pcab-001`** — GERD 차세대 PCAB (한국 K-pharma 기존 케이캡 / 펙수클루 후속).

## 7. 솔직한 한계

⚠️ **모두 컴퓨터 시뮬레이션 단계.** 실제 환자 임상시험은 0건.

⚠️ **양자-VQE 적용 한계도 솔직히 명시:**
- 정책/윤리/디지털 치료제 → 양자 적용 불가 (modality 외부)
- 한의학 multi-component → 단일 분자만 가능
- 세포치료 mfg → wet-lab 필수

⚠️ **많은 후보가 "다음 단계 이미 있는 약 보다 한 단계 더 좋게"** 형식. 진짜 새 mechanism 은 일부.

⚠️ **임상 1상부터 약 10년+, 비용 수천억 원** 필요. 컴퓨터 설계 → 실제 약은 매우 먼 길.

## 8. 결론

**hexa-bio는 "양자 컴퓨터로 신약 후보 200개를 9개월에 설계한 in-silico 프로젝트"** 입니다.

실제 약이 되려면 wet-lab + 임상시험이 필수지만, **양자 계산 layer 가 신약 R&D 의 한 step 으로 자리 잡을 가능성**을 100→200 round-number capstone 두 개로 입증한 셈입니다.
