# ❤️ CARDIO+ — "스텐트 시술 후 재발을 막는 맞춤 방어 패키지"

> 쉽게 읽는 한 장 요약 (easy 버전). 정식 통합 논문: `PAPERS/cardio-plus-postpci/` (10쪽).
> 새 약을 발명한 게 아니라, **이미 있는 약·유전자검사·근거를 환자 맞춤으로 재조립**한 in-silico 근거 패키지입니다(정직).

---

## 한 줄로

- **하는 일**: 심장 스텐트(PCI) 시술을 받은 사람이 **다시 막히거나 나빠지지 않도록**, 네 갈래 위험을 한 번에 막는 맞춤 전략 묶음.
- **별칭**: "스텐트 후 재발 방어 패키지"
- **비유**: 수리한 도로(뚫은 혈관)에 **사고를 막는 4종 안전장치**(가드레일·과속카메라·제설·배수)를 한꺼번에 다는 것. 차(약)는 그대로지만 **어디에·누구에게** 다느냐를 유전자검사로 정밀히 고름.

```
[ 막힌 혈관 ] ──PCI 스텐트──▶ [ 뚫림 ] ──?──▶ 재발 위험 4갈래
                                              ├─ 미세혈관 막힘
                                              ├─ 항혈소판제 안 듣는 체질
                                              ├─ 스텐트 재협착
                                              └─ Lp(a) 고위험 유전
```

---

## 왜 중요한가 — 뚫었다고 끝이 아니다

심장 혈관이 막히면 **스텐트(PCI)** 로 뚫습니다. 그런데 뚫은 뒤에도 **다시 막히거나, 약이 안 듣거나,
숨은 유전 위험 때문에** 재발·합병증이 옵니다. 한국인은 특히 두 유전형이 흔합니다 —
**CYP2C19**(항혈소판제 클로피도그렐이 잘 안 듣는 체질), **ALDH2*2**(술 마시면 얼굴 빨개지는 그 유전자, 혈관보호와 연관).
**"한 명 한 명의 유전형에 맞춰" 방어해야** 재발을 줄일 수 있는데, 현실은 모두에게 같은 처방을 합니다.

---

## 어떤 적용이 가능한가 — 4갈래 방어 + 채혈 한 번

CARDIO+는 post-PCI(스텐트 후) 재발을 **네 방향에서 동시에** 막는 전략을 묶고, 각각을 한국에서
바로 굴릴 임상시험으로 설계했습니다.

```
              [ CARDIO+ 통합 방어 ]
       ┌──────────┬──────────┬──────────┐
    NOREFLOW   DAPT-PGx     ISR        LPA
   미세혈관막힘  항혈소판 맞춤  재협착     Lp(a)낮추기
```

| 갈래 | 무엇을 막나 | 전략 | 시험 우선순위 |
|---|---|---|---|
| 🩸 DAPT-PGx | 항혈소판제 안 듣는 체질 | CYP2C19 유전형 맞춤 처방 | 🥇 **1순위** (새 신약 허가 불필요·즉시 시작) |
| 🫀 NOREFLOW | 미세혈관이 안 열림(no-reflow) | 니코란딜+아데노신 등 | 🥈 2순위 |
| 🧬 LPA | 타고난 Lp(a) 고위험 | Lp(a) 낮추는 siRNA | 🥉 3순위 |
| 🔧 ISR | 스텐트 안이 다시 좁아짐 | 비-mTOR 계열 접근 | 4순위 |

> 🎯 **핵심 한 수 — 채혈 한 번(PCR 패널)으로 6개 벽을 동시에 넘는다.**
> `ALDH2*2 × CYP2C19 × Lp(a)` 세 가지를 **한 번 채혈**로 같이 검사하면, 네 갈래 위험을 한꺼번에
> 분류할 수 있고 **기존 한국 건강보험 급여 안에서 배포 가능**합니다.

```
   기존 방식                  │   CARDIO+ 방식
 ─────────────               │  ─────────────
  모두 같은 처방              │   채혈 1회 → 3축 유전형
  검사 따로따로               │   → 4갈래 맞춤 방어 동시
  − 재발 위험 큼              │   + 보험 안에서 바로 적용
```

## 핵심 발견 (정직하게)

```
전 (막연한 "다 좋다")          후 (CARDIO+ 정직 분류)
─────────────────             ─────────────────
 ▢ 다 효과 있다              →  ▢ 등급으로 솔직히 나눔
                               🔵 강한근거 1 · 🟢 43 · 🟡 인용근거 215
                               🟠 45 · 🔴 닫힌음성 5 · ⚪ 추정 31
```

- **닫힌 음성도 정직하게(🔴 5건)**: no-reflow는 대형 임상 4건이 효과 없음으로 결론 → 헛된 길을 솔직히 표시.
- **즉시 실행 가능한 1순위**: DAPT-PGx(유전형 맞춤 항혈소판)는 **새 약 허가가 필요 없어 바로 시작** 가능.
- **본인 사례**: 시술받은 당사자 10년 시뮬레이션에서 +0.066 QALY(삶의질보정수명) 개선 — 단, 이는 **예시적 추정(⚪)**.
