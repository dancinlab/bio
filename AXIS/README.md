## В§1 The expansion question + 5 promotion criteria

Should there be a 6th axis? A candidate is promotable only if it passes
all five (each scored 0вҖ“1, M/N/R/S/F; 4.5+ AND 5/5 to truly promote):

1. **Modality (not scope)** вҖ” a molecular/drug-action modality, not a
   disease range
2. **Non-overlap < 30%** вҖ” вүҘ30% overlap with an existing axis вҮ’ demote
   to sub-axis
3. **Rigor** вҖ” mechanism + verifiable endpoint + mechanistic publication
4. **Scope discipline (CDER)** вҖ” drug-only; device(CDRH)/biologic-only
   (CBER) avoided or explicitly flagged
5. **Falsifier accessibility** вҖ” an `F-Q-6-*` falsifier is definable

Previously rejected (re-confirmed): BIO-EVOLUTION (borderline) В·
QUANTUM-BIOLOGY (pseudoscience risk) В· PLANETARY-HEALTH (scope mismatch)
В· CONSCIOUSNESS (too vague, no falsifier).
