# CRO evaluation matrix (Phase 1)

**STATUS**: empty template — fill as RFP responses arrive

For each CRO that responds to an RFP, score across these dimensions:

| Dimension | Weight | Scoring rubric |
|---|---|---|
| **Technical capability** | 0.30 | 0=no relevant experience; 5=multiple published projects in same area |
| **Timeline match** | 0.15 | 0=much longer than asked; 5=meets or beats timeline |
| **Budget fit** | 0.20 | 0=2x+ over cap; 5=within or under cap |
| **MTA / NDA terms** | 0.10 | 0=unfavorable IP terms; 5=clean UBMTA-equivalent |
| **References quality** | 0.10 | 0=no refs; 5=multiple recent peer-reviewed pubs from CRO scientists |
| **Geography / logistics** | 0.05 | 0=heavy shipping/import friction; 5=local or low-friction |
| **Communication quality** | 0.10 | 0=slow / unclear replies; 5=responsive + technically deep |

**Total**: weighted sum / 5 → normalized 0-1 score. Select top 1-2 per axis.
